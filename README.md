Este es un sub-software de código abierto codificado en Base64 y Bashscript que ejecuta de forma legible <a href="https://es.wikipedia.org/wiki/Youtube-dl">YT-DLP</a> en Termux en una versión codificada en un formato comprensible generacional para versatilizar el proceso de descarga de música e audio, y contenido audiovisual o de video, e utilizar internamente para conocer de una forma experimental o através de el procesamiento de este Bashscript secuencias de comandos básicos e útiles de YT-DLP y la consecución de procesos multimedia que ofrece beneficiariamente el autor y el programador de <a href="https://es.wikipedia.org/wiki/Youtube-dl">YT-DLP</a> gratuitamente hacia un público general con <a href="https://es.wikipedia.org/wiki/Youtube-dl">YT-DLP</a>, cabe resaltar también al igual que con la misma objetividad de la digitalización de este repositorio y sub-software digitalizado en Bashscript y codificado con secuencias de comandos de <a href="https://es.wikipedia.org/wiki/Youtube-dl">YT-DLP</a> y uno de los software más sofisticado de extracodificación streaming e extracodificación multimedia escrito en Python de código abierto hoy en día, y más reconocido en la actualidad por su eficiencia y funcionalidad llamado en una versión anterior <a href="https://es.wikipedia.org/wiki/Youtube-dl">Youtube-dl</a> y hoy en día <a href="https://pypi.org/project/yt-dlp">YT-DLP</a>.</p>

<br />
~ $ ``` yes | pkg upgrade && yes | pkg update ```
<br />

<br />
~ $ ``` yes | pkg install python ```
<br />
~ $ ``` pip install yt-dlp ```
<br />
~ $ ``` yes | pkg install git ```
<br />
~ $ ``` git clone https://github.com/wolfkrypter/YTERMUX-V2.3.12.git ```
<br />

~ $ ``` cd YTERMUX-V2.3.12 ```
<br />
~/YTERMUX-V2.3.12 $ ``` chmod +x ytermux-v2 ```
<br />

~/YTERMUX-V2.3.12 $ ``` mv ytermux-v2 $PATH ```
